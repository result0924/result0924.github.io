### Web 2.0 vs Web 3.0
web2.0可能需要註冊和驗證身份的方式才能使用<br/>
web3.0則只需要一個錢包就可以做很多事

web 1 去中心化<br/>
web 2 將一切都集中到平台<br/>
web 3 將再次將一切去中心化<br/>
web 3 應該給我們web2的豐富性，但去中心化
